# Skill: Report (Skill 07)

## Objective
Create professional-grade QA reports for stakeholders and developers.

## Instructions
1. **Summary Table:** High-level pass/fail/skip counts.
2. **Execution Context:** Version, environment, and date/time.
3. **Critical Path Status:** Identify if P0/P1 scenarios are green.
4. **Bug List:** Detailed technical reports for every failure.
5. **Risk Assessment:** Final recommendation (e.g., "Ready for Release" or "Blocked").
